# Apple Docs Explorer v0.1.1

Patch release focused on public project branding and app identity.

## What changed

- Added a generated macOS app icon for Apple Docs Explorer.
- Wired the icon into the Xcode asset catalog as `AppIcon`.
- Added a public README/social preview image for repository sharing, LinkedIn posts, and release notes.
- Updated README and screenshot documentation to reference the new brand assets.

## Release assets

This release attaches:

- `apple-docs-explorer-app-icon.png`: 1024px source app icon image.
- `apple-docs-explorer-social-card.png`: 1600x900 public project/social preview image.

## Validation

Validated locally with:

```bash
swift test
./scripts/generate_project.sh
xcodebuild -project AppleDocsExplorer.xcodeproj -scheme AppleDocsExplorer -configuration Release build
```

The Release build emitted `AppIcon.icns` into the app bundle, confirming the asset catalog wiring.

## Notes

No search, MCP, CLI, Docker, or documentation retrieval behavior changed in this patch release.
